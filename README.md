# Twitch VOD Downloader 📺 📥

A lightweight and powerful desktop application to download Twitch VODs effortlessly. Built for stability, it handles even the longest streams (10+ hours) with ease.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-purple.svg)

## ✨ Features
- **All-in-One Standalone**: No need to install Python, FFmpeg, or any other tools. Everything is bundled in a single `.exe` file.
- **High Quality**: Choose your preferred resolution (1080p, 720p, 480p, or Best available).
- **Full Control**: Pause, Resume, or Cancel your downloads at any time.
- **Real-time Stats**: Track download speed, percentage, and estimated time remaining (ETA).
- **Portable**: No installation required. Just download and run.

## 🚀 How to Download and Use
1. Look at the **"Releases"** section on the right side of this page.
2. Download the latest `TwitchDownloader.exe`.
3. Double-click the file to launch the application.
4. Paste the Twitch VOD URL, select your save folder, and hit **DOWNLOAD VOD**.

## 🛡️ Antivirus Notice
Since this application is bundled into a single executable using PyInstaller and is not digitally signed (which costs a lot!), some antivirus software or **Windows SmartScreen** might flag it as a "False Positive." 
- This app is safe and open-to-inspection logic-wise. 
- You may need to click *"More info"* -> *"Run anyway"* to start the program.
