# Twitch VOD Downloader 📥

Twitch Vod Downloader is a simple application that lets you donwload any Twitch Vod you want.
![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Version](https://img.shields.io/badge/version-1.0.0-purple.svg)

##  Features
Twitch Vod Downloader is a portable app (no installation needed) and lets you download any twitch Vod in a folder of your choice while giving you the option to choose the quality of the Vod.

##  How to Download and Use
1. Look at the **"Releases"** section on the right side of this page.
2. Download the latest `TwitchVodDownloader.exe`.
3. Double-click the file to launch the application.
4. Paste the Twitch VOD URL, select your save folder, and hit **DOWNLOAD VOD**.

##  Antivirus Notice
This application is bundled in a single .exe file and lacks a digital signature so some antiviruses may flag this application as a false positive.
